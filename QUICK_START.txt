═══════════════════════════════════════════════════════════════════
                      FIRST TIME SETUP
═══════════════════════════════════════════════════════════════════

iOS (Mac only):
  1. Install Xcode from App Store
  2. Run: sudo gem install cocoapods
  3. Run: ./open-app.sh

Android (Any OS):
  1. Install Android Studio
  2. Tools → Device Manager → Create emulator
  3. Start the emulator
  4. Run: ./open-app.sh (Mac/Linux) or open-app.bat (Windows)
