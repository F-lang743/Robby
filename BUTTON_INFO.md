# 🎯 ONE-BUTTON LAUNCHER INFO

## What is the name of the button?

### On Mac/Linux:
**File name:** `open-app.sh`
**Command to run:** `./open-app.sh`

### On Windows:
**File name:** `open-app.bat`
**Command to run:** `open-app.bat` (or just double-click it)

---

## What does it say when you run it?

When you run the script, here's exactly what you'll see:

```
════════════════════════════════════════════════════════
  🚛 ROBBY TRUCK ASSISTANT - ONE-BUTTON LAUNCHER 🎤
════════════════════════════════════════════════════════

📦 Step 1: Checking dependencies...
   ✅ Dependencies already installed!

🚀 Step 2: Starting Metro bundler...
   Stopping existing Metro...
   Starting Metro bundler...
   Metro starting (PID: 1234)...
   Waiting for Metro to be ready...
   ✅ Metro bundler is ready!

📱 Step 3: Launching app...

Choose your platform:
  1) iOS (requires macOS + Xcode)
  2) Android (requires Android Studio + Emulator/Device)
  3) Just keep Metro running (I'll launch manually)

Enter choice (1, 2, or 3): █
```

---

## What do you type?

After seeing the menu, you type:
- **1** for iOS (Mac only)
- **2** for Android 
- **3** to just keep Metro running without launching

Then press **Enter**!

---

## Summary

**Script names:**
- `open-app.sh` (Mac/Linux)
- `open-app.bat` (Windows)

**Main message it displays:**
```
🚛 ROBBY TRUCK ASSISTANT - ONE-BUTTON LAUNCHER 🎤
```

**What it does:**
1. Checks/installs dependencies
2. Starts Metro bundler
3. Asks which platform
4. Launches the app!

**That's your one button!** 🎉
